# Блогикум

## Описание проекта
Проект включает модели для публикаций, категорий и локаций, панель администратора, ORM-запросы для фильтрации опубликованных постов и шаблоны для отображения контента.

### Основные функции
- **Фильтрация постов**: Отображаются только опубликованные посты с датой публикации не позже текущей и опубликованной категорией.
- **Админ-панель**
- **Страницы**: Главная (5 последних постов), страница категории, детальная страница поста.



## Установка и запуск
1. Клонируйте репозиторий: `git clone https://github.com/JuliaDJ1/django-sprint3.git`.
2. Создайте и активируйте виртуальное окружение: `python -m venv venv` и `venv\Scripts\activate` или `source venv/bin/activate`.
3. Установите зависимости: `pip install -r requirements.txt`.
4. Перейдите в `blogicum/`: `cd blogicum`.
5. Примените миграции: `python manage.py migrate`.
6. Загрузите фикстуры: `python manage.py loaddata ../db.json`.
7. Запустите сервер: `python manage.py runserver`.
8. Доступ: 
   Сайт: http://127.0.0.1:8000/
